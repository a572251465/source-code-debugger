> - 致力于源码断点，一行一行 理解源码，理解源码中的设计思想
> - 通过断点打包后的源码， 带大家了解每一个知识点。

## 1. 合适人群

- 优秀的应届毕业生
- 对源码方面执着的人
- 工作 3~4+ 年的开发人员

## 2. 目的

- 对自我源码学习的一种积累
- 提高项目`知名度`
- 知识共享，免费学习
